La división es la operación matemática inversa
a la multiplicación. Cosiste en encontrar
cuántas veces está contenido un número en
otro. En ocasiones se reparten objetos en
cantidades iguales. Las partes que componen a
una división son:
2
5 10
0
1.
Los pasos a seguir para realizar una
división:
2.
Se empieza desde la izquierda.
3.
Se reparten las cifras del dividendo entre
las del divisor.
4.
Se divide utilizando las tablas de
multiplicar al revés.
5.
Se multiplica la cifra del cociente por el
divisor y se resta del dividendo. Si no se
puede restar se prueba con un número
menor.
6.
Se toma la siguiente cifra del dividendo
inicial y se repite este proceso hasta
haber tomado todas las cifras.
